与jQuery的混用

有三种方案，第一种，使用avalon自带的加载器引用，第二种禁用avalon自带加载器，使用requirejs引用，第三种禁用avalon自带加载器，直接引用。

第一种，将avalon与jquery放在同一目录下，然后：

<!DOCTYPE html>
<html>
    <head>
        <title></title>
        <script src="avalon.js" type="text/javascript"></script>
        <script type="text/javascript">
            require.config({
                paths: {
                    jquery: "jquery.js",
                    avalon: "avalon.js"
                },
                shim: {
                    jquery: {
                        exports: "jQuery"
                    },
                    avalon: {
                        exports: "avalon"
                    }
                }
            })
            require(["jquery", "avalon"], function($, avalon) {
                console.log($)
                console.log(avalon)
            })
        </script>
    </head>
    <body ms-controller="test">
        TEST
    </body>
</html>

修改avalon源码 loader: ture ---> loader: false，然后
